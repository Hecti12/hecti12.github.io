# hecti12.github.io

Personal GitHub Pages portfolio for Hector Enriquez Orellana.

The site presents IT support, help desk, systems administration, and business analytics projects in one static portfolio hosted at:

https://hecti12.github.io

## Structure

- `index.html` - portfolio content and sections
- `styles.css` - responsive visual design
- `app.js` - mobile navigation, active section state, and intro typing effect
- `assets/headshot.png` - professional headshot used by the homepage and sidebar
- `assets/project-*.png` - project images used by the IT and analytics cards

## Update Notes

When new project repositories are ready, add them to either the `#it-projects` or `#analytics-projects` section in `index.html`.

The layout is inspired by the section structure of Varad Bhogayata's MIT-licensed portfolio template, adapted with custom styling, copy, assets, and project content.
