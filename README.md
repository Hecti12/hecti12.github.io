# hecti12.github.io

Personal GitHub Pages portfolio for Enrique Hecti.

The site presents IT support, help desk, systems administration, and business analytics projects in one static portfolio hosted at:

https://hecti12.github.io

## Structure

- `index.html` - portfolio content and sections
- `styles.css` - responsive visual design
- `app.js` - project category filtering
- `assets/hero-workspace.png` - generated hero image used by the homepage

## Update Notes

When new project repositories are ready, add them as new `.project-card` entries in `index.html` and set `data-category` to `it`, `analytics`, or both.
